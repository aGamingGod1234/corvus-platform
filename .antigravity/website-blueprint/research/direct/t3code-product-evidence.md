## Product facts used by the design

- T3Code describes itself as a minimal web GUI for coding agents.
- Its documented provider set includes Codex, Claude, Cursor, and OpenCode.
- It supports a desktop application as well as the web-oriented interface.
- Provider readiness is explicit: users install and authenticate a supported provider before use.

## Adaptation boundary

Corvus may use the product-level lesson that provider context and readiness belong close to agent work, and that one surface can remain provider-neutral. Corvus does not copy T3Code application chrome, layout geometry, theme, navigation, commands, provider presentation, brand assets, or wording.
